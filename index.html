Мои фотоработы

Проекты

Фотогалерея

Этот проект представляет собой многостраничный сайт с галереей с плавающими фотографиями (с использованием Draggable и InertiaPlugin). Реализован прелоадер, свайпер с таймером, смена цвета темы.

Форма обратной связи

Этот проект представляет собой форму обратной связи — своеобразный инструмент для совершенствования образовательного процесса и взаимодействия между студентами и преподавателями.

Счётчик

Этот проект — простой счётчик, позволяющий изменять числовое значение (с разным шагом), отображаемое на экране. Есть кнопка для сброса и возможность управлять с помощью горячих клавиш.

Генерация случайной фотографии

Проект предназначен для получения случайных изображений собак из внешнего API с использованием асинхронных методов на JavaScript. Полученные изображения отображаются на веб-странице в качестве развлечения для пользователей, есть возможность добавлять фотографии в "Избранное" и удалять их с помощью горячих клавиш.

Форма регистрации

Проект демонстрирует проверку формы на стороне клиента с использованием JavaScript. В процессе выполнения проверяются данные, введенные пользователем в форму, перед отправкой. При успешной проверке отображается сообщение об успехе, а введенные данные становятся доступными для просмотра в режиме только для чтения.

Игра (новелла) в стиле LEGO

Этот проект представляет собой визуальную новеллу (жанр компьютерных игр, подвид текстового квеста). В данной игре несколько вариантов развития сюжета, всё зависит только от выбора того или иного действия. Во время прохождения этой новеллы на фоне играет захватывающая музыка.

Генерация случайных шуток

Проект использует внешний API для получения случайных шуток с использованием асинхронных методов на JavaScript. После получения данные отображаются на веб-странице в виде шутки для развлечения пользователей. Есть возможность скопировать шутку в буфер обмена.
